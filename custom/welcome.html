FHIR 臺灣公開測試伺服器（R4 版本 / TW Core IG 臺灣核心規範 / 開啟驗證器)


歡迎來到 FHIR 臺灣公開測試伺服器，本伺服器為測試用途的 FHIR Release R4 版本伺服器，您可以隨意利用，但請不要真的拿來用在生產環境。本伺服器內建 臺灣核心實作指引 (TW Core IG) 並啟用驗證器 (Validator)，可用於測試衛福部電子病歷交換情境。若您是想尋找沒有套用實作指引 (IG) 的國際總會版本 FHIR 伺服器，請造訪這個網站。

此伺服器使用開放原始碼的 HAPI FHIR 專案，並使用 FHIR.tw 臺灣 FHIR 公共網域名稱服務，這是一個專供與 FHIR 標準與醫學資訊相關專案或學術研究使用的免費社群服務。
